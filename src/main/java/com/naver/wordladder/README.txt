문제

시작단어와 목표단어가 주어집니다. 시작단어에서 한글자씩 임의로 바꿀 수 있습니다. 단, 그 단어는 주어진 사전에 있어야 하는 단어입니다.

가장 짧은 경로를 찾았을 때 거쳐간 단어수를 출력하세요

아래 예에서 가장 짧은 경로는 "hit" -> "hot" -> "dot" -> "dog" -> "cog" 이므로 프로그램은 시작부터 목표까지 총 거쳐간 단어수 '5'를 출력해야합니다.

입력
start = "hit"
end = "cog"
dict = ["hot","dot","dog","lot","log"]

출력
5

해결방법
Queue 및 트리를 이용해서 해결

시작단어를 노드로 만들고 그 노드를 루트로 만든다.
루트를 큐에 넣고 아래를 반복한다.
큐에서 노드를 하나 빼서 1Distance단어들을 찾아내 노드로 만든다.
이 노드의 단어와 마지막단어가 1Distance이고 shortestPath보다 작다면 shortestPath를 설정하고 출력한다.
큐에 노드를 넣는다.
